## 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 标签名称 | 文本 | 是 | 标签名称，支持中文和英文 |
| 标签颜色 | 颜色 | 否 | 从预设颜色选择或自定义 |

## 交互流程

### 1. 打开对话框
- 点击标签管理页面的「添加标签」按钮
- 在项目添加对话框中输入新标签名称

### 2. 输入标签名称
- 输入标签名称
- 实时验证标签名称唯一性

### 3. 选择颜色
- 从预设颜色中快速选择
- 或使用颜色选择器自定义

### 4. 预览
- 实时显示标签效果

### 5. 保存
- 验证名称唯一性
- 保存到数据库
- 刷新标签列表

## 验证规则

- 标签名称：不能为空，长度1-50字符
- 标签名称：同一用户下唯一
- 标签颜色：默认为蓝色 (#3498db)

## 预设颜色

| 颜色 | 代码 | 说明 |
|------|------|------|
| 🔴 红色 | #e74c3c | 紧急、重要 |
| 🟠 橙色 | #f39c12 | 警告、注意 |
| 🟡 黄色 | #f1c40f | 待处理 |
| 🟢 绿色 | #2ecc71 | 完成、正常 |
| 🔵 蓝色 | #3498db | 默认、通用 |
| 🟣 紫色 | #9b59b6 | 特殊、高级 |
| ⚫ 黑色 | #34495e | 稳定、基础 |
| ⚪ 白色 | #ecf0f1 | 轻量、简洁 |
