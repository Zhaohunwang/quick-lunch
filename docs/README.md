## 项目简介

Project Hub是一个跨平台的统一项目管理工具，用于管理多个IDE项目，支持快速搜索、标签分类、工作区管理，并提供MCP接口供AI工具调用。

### 核心功能

- 统一项目管理入口
- 快速搜索与访问
- 多IDE支持与配置
- 标签系统与工作区管理
- AI集成（MCP协议）

### 技术栈

- 前端：Avalonia UI 11.x
- 运行时：.NET 8 (LTS)
- 语言：C# 12
- 数据库：SQLite
- 架构：MVVM
