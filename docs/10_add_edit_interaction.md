### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 项目名称 | 文本 | 是 | 项目的显示名称 |
| 项目路径 | 文本 | 是 | 支持浏览选择文件夹 |
| 别名 | 文本 | 否 | 快速访问的短名称 |
| 描述 | 文本 | 否 | 项目描述信息 |
| 标签 | 下拉多选 | 否 | 选择已有标签，或输入创建新标签 |
| 默认IDE | 下拉单选 | 否 | 从已配置的IDE列表中选择 |
| 项目颜色 | 颜色 | 否 | 自定义项目显示颜色 |
| 收藏 | 布尔 | 否 | 是否收藏项目 |

### 交互流程

1. **打开对话框**
   - 点击主界面 `[+]` 按钮
   - 右键菜单选择「添加项目」

2. **选择路径**
   - 点击浏览按钮选择文件夹
   - 或直接输入路径

3. **配置标签**
   - 点击标签下拉框
   - 显示已有标签列表（多选）
   - 可直接输入新标签名称（回车创建）
   - 点击已选标签的×可移除

4. **选择默认IDE**
   - 从下拉列表选择一个IDE
   - 列表来自全局IDE配置

5. **保存**
   - 验证必填字段
   - 保存到数据库
   - 刷新项目列表

### 交互流程

1. **打开对话框**
   - 在设置页面点击「添加IDE」

   - 点击「从模板选择」
   - 从预设IDE模板中选择
   - 自动填充路径和参数

3. **手动配置**
   - 输入IDE名称
   - 浏览选择可执行文件
   - 可选输入启动参数

4. **测试启动**
   - 点击「测试启动」
   - 验证配置是否正确

5. **保存**
   - 验证路径有效性
   - 保存配置

## 验证规则

### 项目验证
- 项目名称：不能为空，长度1-100字符
- 项目路径：必须是有效目录，且可访问
- 别名：长度0-50字符，同一用户下唯一
- 标签：最多10个标签

### 标签验证
- 标签名称：长度1-20字符，全局唯一
- 标签颜色：有效的十六进制颜色值

### 工作区验证
- 工作区名称：不能为空，长度1-100字符
- 项目选择：至少选择一个项目
- 项目数量：最多20个项目

### IDE验证
- IDE名称：不能为空，长度1-50字符
- 可执行文件路径：必须是有效文件，且可执行
- 启动参数：长度0-200字符
