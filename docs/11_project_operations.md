# 项目操作功能设计

## 项目删除

### 删除确认

```
┌─────────────────────────────────────────────────────────────┐
│  确认删除                                    [-] [□] [×]       │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ⚠️ 确定要删除项目 "电商平台重构" 吗？                         │
│                                                              │
│  此操作将：                                                   │
│  • 从项目列表中移除该项目                                      │
│  • 保留项目在磁盘上的文件和文件夹                               │
│                                                              │
│  ☐ 同时删除项目关联的工作区引用                                │
│                                                              │
│                                                              │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │ [取消]                         [确认删除]                 │ │
│  └─────────────────────────────────────────────────────────┘ │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 删除操作入口

1. **项目列表操作按钮**
   - 每个项目行右侧显示删除按钮 🗑️
   - 点击后弹出确认对话框

2. **右键菜单**
   - 在项目上右键 → 删除项目

### 删除流程

1. **触发删除**
   - 点击删除按钮

2. **确认对话框**
   - 显示项目名称
   - 说明删除影响范围
   - 提供取消和确认选项

3. **执行删除**
   - 从数据库删除项目记录
   - 更新项目列表显示
   - 更新统计数据（全部、收藏、最近数量）

4. **删除后处理**
   - 刷新左侧导航栏数量
   - 如果当前在标签筛选视图，刷新筛选结果

### 删除限制

- 不删除磁盘上的实际文件
- 不删除关联的标签（标签独立管理）
- 可选：是否从工作区中移除该项目引用

## 项目编辑

### 编辑入口

1. **项目列表操作按钮**
   - 每个项目行右侧显示编辑按钮 ✏️
   - 点击后打开编辑对话框

2. **右键菜单**
   - 在项目上右键 → 编辑项目

### 编辑对话框

编辑对话框复用添加项目对话框，区别在于：

1. **标题变化**
   - 添加模式：标题显示 "添加项目"
   - 编辑模式：标题显示 "编辑项目"

2. **数据预填充**
   - 自动加载项目现有数据
   - 项目名称、路径、别名、描述
   - 已选标签
   - 默认IDE
   - 收藏状态

3. **保存逻辑**
   - 保留项目ID和创建时间
   - 更新修改时间为当前时间
   - 更新数据库记录

### 编辑流程

1. **打开编辑对话框**
   - 点击编辑按钮
   - 对话框标题显示 "编辑项目"
   - 自动填充现有数据

2. **修改字段**
   - 修改需要更新的字段
   - 支持修改所有字段（包括路径）

3. **保存更改**
   - 验证必填字段
   - 更新数据库记录
   - 刷新项目列表显示
   - 更新统计数据

### 字段编辑限制

| 字段 | 可编辑 | 说明 |
|------|--------|------|
| 项目名称 | ✅ | 随时可修改 |
| 项目路径 | ✅ | 支持修改路径 |
| 别名 | ✅ | 随时可修改 |
| 描述 | ✅ | 随时可修改 |
| 标签 | ✅ | 可添加/删除标签 |
| 默认IDE | ✅ | 可修改默认IDE |
| 收藏状态 | ✅ | 可切换收藏状态 |
| 创建时间 | ❌ | 不可修改，仅显示 |
| 修改时间 | ❌ | 自动更新为当前时间 |

## 项目添加

### 添加入口

1. **主界面按钮**
   - 点击右上角 `+` 按钮
   - 打开添加项目对话框

2. **右键菜单**
   - 在项目列表空白处右键 → 添加项目

### 添加流程

1. **打开添加对话框**
   - 对话框标题显示 "添加项目"
   - 所有字段为空

2. **填写信息**
   - 项目名称（必填）
   - 项目路径（必填）
     - 点击"文件夹"按钮选择文件夹
     - 点击"文件"按钮选择文件
     - 选择文件时，IDE选项自动禁用
   - 别名（可选）
   - 描述（可选）
   - 标签（可选）
     - 输入标签名称后点击"添加"
     - 点击标签上的×可删除
   - 默认IDE（文件夹时可选）
   - 收藏状态（可选）

3. **保存项目**
   - 验证必填字段
   - 保存到数据库
   - 刷新项目列表
   - 更新统计数据
   - 更新标签列表

### 路径选择

支持两种路径类型：

1. **文件夹路径**
   - 点击"📁 文件夹"按钮
   - 打开文件夹选择对话框
   - 选择项目根目录
   - IDE功能可用

2. **文件路径**
   - 点击"📄 文件"按钮
   - 打开文件选择对话框
   - 选择特定文件
   - IDE功能自动禁用
   - 显示提示："选择文件时无法使用IDE打开"

## 操作反馈

### 成功反馈

- 添加成功：显示成功提示，项目出现在列表中
- 编辑成功：显示成功提示，项目信息更新
- 删除成功：显示成功提示，项目从列表中移除

### 错误处理

- 路径不存在：提示"路径不存在，请检查后重试"
- 名称重复：提示"项目名称已存在"
- 保存失败：提示"保存失败，请重试"
