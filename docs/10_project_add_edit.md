## 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 项目名称 | 文本 | 是 | 项目的显示名称 |
| 项目路径 | 文本 | 是 | 支持浏览选择文件夹或文件 |
| 别名 | 文本 | 否 | 快速访问的短名称 |
| 描述 | 文本 | 否 | 项目描述信息 |
| 标签 | 文本输入 | 否 | 输入标签名称后点击添加 |
| 默认IDE | 下拉单选 | 否 | 从已配置的IDE列表中选择 |
| 项目颜色 | 颜色 | 否 | 自定义项目显示颜色 |
| 收藏 | 布尔 | 否 | 是否收藏项目 |

## 交互流程

### 1. 打开对话框
- 点击主界面 `[+]` 按钮
- 右键菜单选择「添加项目」

### 2. 选择路径
- 点击"📁 文件夹"按钮选择文件夹
- 点击"📄 文件"按钮选择文件
- 或直接输入路径

### 3. 配置标签
- 输入标签名称
- 点击"添加"按钮添加标签
- 点击已选标签的×可移除

### 4. 选择默认IDE
- 从下拉列表选择一个IDE
- 列表来自全局IDE配置
- 选择文件时，IDE选项自动禁用

### 5. 保存
- 验证必填字段
- 保存到数据库
- 刷新项目列表

## 路径选择

### 文件夹路径
- 点击"📁 文件夹"按钮
- 打开文件夹选择对话框
- 选择项目根目录
- IDE功能可用

### 文件路径
- 点击"📄 文件"按钮
- 打开文件选择对话框
- 选择特定文件
- IDE功能自动禁用
- 显示提示："选择文件时无法使用IDE打开"

## 验证规则

- 项目名称：不能为空，长度1-100字符
- 项目路径：必须是有效路径，且可访问
- 别名：长度0-50字符
- 标签：最多10个标签
